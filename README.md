🌦️ Weather Dashboard
📖 Overview

The Weather Dashboard is a web application built using HTML, CSS, and JavaScript that provides users with real-time weather information for any city. By integrating a weather API, the application displays current weather conditions such as temperature, humidity, wind speed, atmospheric pressure, and weather descriptions through a clean and interactive interface.

This project was developed to strengthen front-end development skills by working with APIs, asynchronous JavaScript, DOM manipulation, and responsive web design. It offers a fast and user-friendly way to access accurate weather updates from anywhere in the world.

✨ Features
Search weather by city name
Real-time weather information
Current temperature display
Weather condition (Sunny, Cloudy, Rainy, etc.)
Humidity information
Wind speed display
Atmospheric pressure details
Weather icons based on conditions
Responsive design for desktop and mobile devices
Clean and user-friendly interface
Error handling for invalid city names
🛠 Technologies Used
HTML

HTML is used to create the structure of the application, including the search bar, weather information cards, buttons, and layout.

CSS

CSS is used to design the interface with responsive layouts, modern styling, colors, typography, and animations for an enhanced user experience.

JavaScript

JavaScript handles user interactions, fetches weather data from the API, processes responses, updates the interface dynamically, and manages error handling.

Weather API

A weather API is used to retrieve real-time weather data, including temperature, humidity, wind speed, pressure, and weather conditions.

🚀 How It Works
Open the Weather Dashboard application.
Enter the name of a city in the search box.
Click the search button.
The application sends a request to the Weather API.
Weather data is fetched and displayed instantly.
Users can search for different cities to view updated weather information.
